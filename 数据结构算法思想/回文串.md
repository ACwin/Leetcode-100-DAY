## 1.回文串是什么？
> 回文串：一串正着读和反着读都是一样的一种特殊字符串

## 2. 回文串判断方法：

**题型1：判断是否为回文数？**

```C++
//双指针方法：
  bool isPalindrome(const string& s, int start, int end) {
     for (int i = start, j = end; i < j; i++, j--) {
         if (s[i] != s[j]) {
             return false;
         }
     }
     return true;
 }

```
